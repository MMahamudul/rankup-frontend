# RankUp

RankUp is a modern, role-based contest management platform where creators host contests, users participate and win rewards, and admins ensure platform quality and security.The platform is designed with a clean UI, smooth user experience, and scalable backend architecture.

---

##  Live Website
 https://Rankup.web.app  


---

##  Key Features

-  **Secure Authentication & Authorization**
  - Email/password authentication with JWT protection
  - Role-based access control (Admin, Creator, User)

- **Three Distinct User Roles**
  - **Admin**: Manages users, contests, and approvals
  - **Contest Creator**: Creates contests, reviews submissions, declares winners
  - **Normal User**: Joins contests, submits entries, tracks wins

-  **Contest Creation & Management**
  - Creators can create contests with categories, deadlines, prizes, and images
  - Admin approval system for quality control

- **Payment Integration**
  - Secure payment system for contest registration
  - Payment success tracking and participation validation

- **Role-Based Dashboards**
  - Dedicated dashboards for Admin, Creator, and User
  - Real-time data updates using React Query

- **Winner Declaration & Highlight System**
  - Creators declare winners after contest deadline
  - Winners automatically appear in advertisements and user profiles

- **Leaderboard & Popular Contests**
  - Leaderboard ranking users by number of contest wins
  - Popular contests sorted by highest participation count

- **Advanced Search & Filtering**
  - Search contests by category and keywords
  - Filter contests dynamically on the frontend

- **Fully Responsive Design**
  - Optimized for mobile, tablet, and desktop devices
  - Modern UI built with Tailwind CSS

- **Performance Optimized**
  - React Query for caching and automatic refetching
  - Pagination for large datasets (users, contests)

- **Modern & Cozy UI/UX**
  - Animated sections using Framer Motion
  - Clean dashboards, cards, and interactive components
